# # Tokopedia 데이터셋 샘플

<h2>1001개의 Tokopedia 제품으로 구성된 샘플 データセット</h2>

![Tokopedia dataset header](https://github.com/bright-kr/Tokopedia-dataset-samples/blob/main/Tokopedia-datasets.png)

1000개 이상의 레코드로 구성된 Tokopedia データセット 샘플입니다. データセット은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>Tokopedia データセット에 포함된 일부 데이터 포인트:</h2>

* ```product_id```: 제품 ID
* ```title```: 제품 제목
* ```url```: 제품 리스팅 URL
* ```currency```: 가격 통화
* ```delivery```: 제품 배송 세부 정보
* ```final_price```: 제품 최종 가격
* ```initial_price```: 제품 초기 가격
* ```seller_name```: 제품 판매자 이름
* ```description```: 제품 설명
* ```availability```: 제품 재고 상태
* ```reviews_count```: 제품 리뷰 수
* ```rating_count```: 제품을 평가한 사람 수
* ```rating```: 제품 평점
* ```discussion_count```: 제품 토론 수
* ```categories```: 제품 카테고리
* ```images_count```: 제품 이미지 수

그 외에도 더 많은 항목이 있습니다.

이는 "Tokopedia products" データセット에서 파생된 샘플 하위 집합이며,
해당 データセット에는 <b>120만 개 이상의 레코드</b>가 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, 또는 Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 방식 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, 또는 Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 보강: <b>요청에 따라 제공됩니다.</b>

<b>[전체 Tokopedia データセット 받기](https://brightdata.co.kr/products/datasets/tokopedia)</b>.

<h2>Tokopedia データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. Product Insights & Pricing Strategy</h3>
Tokopedia의 최상위 판매 제품을 파악하여 재고, 가격, 및 공급망 전략을 강화할 수 있습니다. 인기 검색어와 수요가 높은 품목에 대한 인사이트를 활용하여 마케팅 활동을 정교화하고 시장에서 경쟁 우위를 유지할 수 있습니다.

<h3>2.  Competitive Analysis</h3>
Tokopedia의 상위 판매자들의 제품 구성, 고객 리뷰, 및 프로모션 전략을 분석하여 그들을 앞서 나갈 수 있습니다. 새로운 비즈니스 기회를 발굴하고 마켓플레이스 내 입지를 강화하는 데 필요한 인사이트를 확보할 수 있습니다.

<h3>3. Consumer Sentiment & Brand Trends</h3>
Tokopedia에서 트렌드로 떠오르는 카테고리와 브랜드를 추적하여 소비자 수요 변화에 대한 정보를 최신 상태로 유지할 수 있습니다. 지역별 제품 인기도를 분석하여 변화하는 구매자 선호에 맞추어 브랜드 전략을 정렬할 수 있습니다.
